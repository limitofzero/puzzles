#Инструкция по установке:

#Билд фронт части
cd path_to_project
cd static/
npm install
npm run build

#сервер
cd ../
pip install -r requirements.txt
python3 app.py

Тестировал на ubuntu в браузере chrome 66. На других работу не проверял.

P.S. Работа фронта сделана таким образом, что ее можно запустить отдельно через dev-server. Для этого в папке
static надо вызвать npm run start:test
Если фронт не может получить json от сервера, то он берет его напрямую(специально так сделал, чтобы можно было работать без сервера)
